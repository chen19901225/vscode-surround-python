# surround-python README

this is a surround extension for python

## Features

surround commands includes: `if`, `ifElse`, `tryFinally`, `tryExcept`, `tryFinallyExcept`, `for`, `with`

keybinding: `alt+shift+z`


## Requirements

* language: python 
* vscode: >= 1.32.0
